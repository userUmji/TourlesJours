# :크루아상: 뚜레쥬르 웹사이트 리뉴얼
뚜레쥬르 브랜드의 공식 웹사이트를 리디자인하고, 사용자 경험을 고려한 반응형 웹사이트로 구현한 프로젝트입니다.
![썸네일](./screenshot.png) <!-- 썸네일 이미지 넣을 수 있다면 좋음 -->
---
## :압정: 프로젝트 개요
- **프로젝트명**: 뚜레쥬르 웹사이트 리뉴얼
- **목적**: 기존 사이트의 불편한 UX/UI 개선, 브랜드 이미지 강화
- **작업기간**: 2025.05 ~ 2025.06
- **사용기술**: `HTML5`, `SCSS`, `JavaScript`, `Swiper.js`, `Font Awesome`, `Figma`(디자인)

## :망치와_렌치: 주요 기능
- 메인 비주얼 영역 Swiper 슬라이더 적용
- 모바일 하단 고정 TOP 버튼
- 브랜드 섹션 이미지 오버레이 + 텍스트 애니메이션
- 메뉴 섹션 자동 슬라이드 및 상세정보 동기화
- 반응형 (767px)
