[ 프로젝트 명 ]  
뚜레쥬르 웹사이트 리뉴얼 프로젝트  

[ 프로젝트 설명 ]  
이 프로젝트는 기존 브랜드 "뚜레쥬르"의 리뉴얼 웹사이트이며, 
브랜드 소개, 제품, 날씨 기반 추천 메뉴, 가맹점 찾기, 고객센터 등의 콘텐츠를 포함하고 있습니다.  

[ 폴더 구조 설명 ]  
├── css/              👉 컴파일된 CSS 파일 저장  
├── scss/             👉 SCSS 소스 파일 저장 (스타일 작업 원본)  
├── js/               👉 JavaScript 파일 저장  
├── images/           👉 이미지 파일 저장  
├── 빵 배경제거/       👉 배경 제거된 이미지 리소스  
├── .git/             👉 Git 버전 관리용 폴더 (깃 사용 시)  

[ HTML 페이지 설명 ]  
- index.html           👉 메인 페이지 (브랜드 전체 요약 및 대표 콘텐츠 포함)  
- brand.html           👉 브랜드 철학 및 연혁 소개  
- product.html         👉 제품 소개 및 메뉴별 상세 보기  
- weather.html         👉 날씨 기반 추천 메뉴 페이지  
- franchise.html       👉 가맹점 소개 및 창업 안내  
- findStore.html       👉 매장 찾기 (지도 또는 검색 기능 포함 예상)  
- CustomerService.html 👉 고객센터 (자주 묻는 질문, 문의 접수 등)  

[ 사용 기술 스택 ]  
- HTML5, SCSS(CSS), JavaScript  
- Swiper.js (슬라이더 기능 구현)  
- Git (버전 관리용)  

[ 실행 방법 ]  
1. 이 프로젝트 폴더를 웹 서버 루트 또는 로컬에 저장  
2. `index.html`을 브라우저로 열어 시작  
(특별한 빌드 도구 없이 정적 HTML만으로 실행 가능)  

[ 기타 ]  
- 이미지 경로 및 스타일은 상대경로 기준입니다.  
- SCSS 사용 시 `scss/` 내 파일을 `css/`로 컴파일하여 사용해야 합니다.
